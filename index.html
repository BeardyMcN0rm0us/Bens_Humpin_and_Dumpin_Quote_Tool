How to get an instant estimate

  1. Pick a job type from the dropdown.
  2. Enter addresses if asked (postcodes work great).
  3. Add extras: stairs, two‑person, waste, or items to build.
  4. Tap Calculate Quote.
  5. Review the breakdown. You’ll see Total journey and Charged mileage.
  6. Tap Send to WhatsApp to message Ben with everything filled in.

Job types (what’s charged)

Tip Run — Collection → Waterbeach. If collection is >50 miles from home, mileage is charged from home. Minimum disposal fee based on waste type.

House Move / FB Pickup / Student — Home → Pickup → Delivery (one‑way charged). Stairs and two‑person if needed.

Emergency Shop Run — Home → Shop → Delivery. Before/after 10pm base fee.

IKEA Collect & Build — Pick a store (sets collection), add items as needed. Mileage: Home → IKEA → Delivery.

Flat Pack Build Only — Enter your address. If it’s over 15 miles from Ben’s home, mileage is charged one‑way (home → destination), rounded up. Otherwise no mileage.

What the numbers mean

  • Total journey — Ben’s full loop (e.g., Home → Destination → Home).
  • Charged mileage — The miles billed @ your per‑mile rate, based on the job rules.
  • Assembly — Time‑based at £/hour (or per‑item, if Ben flips the switch).
  • Estimated total — A range like £80–£100. Final price confirmed after photos/access.
